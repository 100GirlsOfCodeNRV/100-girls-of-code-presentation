100 Girls of Code

And why it's important to get involved

Created by Heather Price

Stats first!

Between 15-17% of Software Engineering jobs are held by women. It is also projected that software eingineering jobs will grow by 22%. It is also shown that 74% of girls are interested in STEM degrees, but only 3% actually pursue them.

So if we want to grow the industry, so need to get more girls to pursue Computer Science Degrees. They need to get excited about it.

100 Girls of Code

This is program designed to get girls interested in Computer Science by providing them with exciting workshops.

These workshops are run by women to help the girls understand that there are girl Software Engineers and this isn't just a "boy's game".


This program provides:

  • Materials for the one-day workshop
  • Teacher Training
  • Promotional Assistance
  • Technical and Education Support
  • Participant Eligibility for Scholarships
  • And more...

Benefits

If Modea were to help sponsor this program we would become more active in the community as well as teach girls that they can do what we do everyday.

This could lead to potiental internship from the participants, better publicity, creating challenges for the current employees of Modea by allowing them to help girls solve problems, and introducint designers to the world of code.
