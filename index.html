The Major Divide

According to a Google doc that collects data on the number of women in tech companies, an averate of 15.73% of tech jobs are held by women. According to a study conducted by CRA-W only about

http://qz.com/143967/the-tech-industrys-woman-problem-statistics-show-its-worse-than-you-think/ - % of women in tech companies

Current at Apple only 20% of their employees are female, Google lags behind them with only 17%.

It is believed this is caused by workplace sexism, a lack of female role models, stereotypes regarding women’s innate technical incompetency and the difficulties of combining tech careers with motherhood.

So if we want to grow the industry, so need to get more girls to pursue Computer Science Degrees. We need to help girls in our community understand that there is so much oppertunity for them.

100 Girls of Code

This is program designed to get girls interested in Computer Science by providing them with exciting workshops.

These workshops are run by women to help the girls understand that there are girl Software Engineers and this isn't just a "boy's game".


100 Girls of Code is a program of TN Code Academy and the Upper Cumberland Entrepreneurial Foundation, a public 501(c)(3) nonprofit organization.

This program provides:

  • Materials for the one-day workshop
  • Teacher Training
  • Promotional Assistance
  • Technical and Education Support
  • Participant Eligibility for Scholarships
  • And more...

Benefits

This program will benefit the community as well as Modea.

Benefits for the community

The program not only introduces girls to the world of computer science, but we feel that it helps girls gain confidence in saying "Hey you can make a difference and you can have fun doing it."

This also encourgages the community to see Modea as a source of education and caring individuals.

The local universities will get an oppertunity to get their CS and Design girls involved in some volunteer work that means something to them.

Benefits for Modea

Modea will be seen in a new light by the media by getting involved in a nationwide issue that helps our futures.

This will also get us into schools and help us recruit more talent as these girls get into universities.

The current Developers will be challenged as teaching helps you see problems in a new light. This will also challenge designers who want to learn to code by giving them the oppertunity to gain some training.

We also feel that collaboration in this program will bleed into the work environment.
